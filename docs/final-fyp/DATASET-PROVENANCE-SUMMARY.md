## 1) Resume NER datasets

### Public / external datasets used


| Source                                                  | Type                                | Notes                                                                                 |
| ------------------------------------------------------- | ----------------------------------- | ------------------------------------------------------------------------------------- |
| DataTurks `entity_recognition_in_resumes.json`          | Public annotated resume NER dataset | Used as an existing base source in the merge pipeline (`prepare_data.py --existing`). |
| Dotin resume dataset (`resume-dataset-NER-annotations`) | Public GitHub dataset               | XML CV annotations; mapped into unified 6-entity schema.                              |
| `vrundag91/Resume-Corpus-Dataset`                       | Public GitHub dataset               | Label Studio-style JSON; supported and mapped by merge script.                        |
| `minhquan23102000/RESUME_NER_DATASET`                   | Public GitHub dataset               | spaCy-style JSON; supported and mapped by merge script.                               |


### Project-generated datasets


| Source                                                                                           | Type                                | Notes                                                                   |
| ------------------------------------------------------------------------------------------------ | ----------------------------------- | ----------------------------------------------------------------------- |
| `generate_resumes_llm.py` outputs (e.g. `llm_generated_resumes.jsonl`, Sri Lankan tech variants) | Synthetic data generated by project | OpenAI-generated resumes in same JSONL schema, merged with public data. |
| Merged files (e.g. `merged_1030_plus_all_llm_plus_proper.json`)                                  | Curated project dataset             | Final merged + cleaned training file used in the frozen run.            |


### Frozen run summary (resume NER)

- Corpus size: **4738**
- Split: **3790 / 473 / 475** (train / val / test, seed 42)
- Test micro F1: **0.83**

## 2) Job-poster NER datasets

### Public / external datasets used


| Source                                       | Type                           | Notes                                                                                                      |
| -------------------------------------------- | ------------------------------ | ---------------------------------------------------------------------------------------------------------- |
| SkillSpan (NAACL 2022, `larkinbb/skillspan`) | Public job-posting NER dataset | Downloaded and converted via `scripts/download_skillspan.py`; skill/knowledge mapped to `SKILLS_REQUIRED`. |


### Public references considered / optionally usable


| Source                                   | Type                    | Notes                                                                                                      |
| ---------------------------------------- | ----------------------- | ---------------------------------------------------------------------------------------------------------- |
| LREC 2022 Job Description Corpus         | Public benchmark corpus | Mentioned in pipeline docs; supports mapping for Occupation/Qualification/Experience/Domain if integrated. |
| O*NET / ESCO / HF job-description tables | Public resources        | Useful as ontology or raw text context, not direct token-level NER labels by default.                      |


### Project-generated datasets


| Source                                                                                                       | Type                                | Notes                                                          |
| ------------------------------------------------------------------------------------------------------------ | ----------------------------------- | -------------------------------------------------------------- |
| `generate_job_postings_llm.py` outputs (e.g. `llm_generated_job_postings.jsonl`, `llm_sri_lanka_jobs.jsonl`) | Synthetic data generated by project | Used to cover entity types beyond SkillSpan-only skill labels. |
| Merged files (e.g. `merged_job_poster_ner_full_varied.json`)                                                 | Curated project dataset             | Combined public + synthetic data used in the frozen run.       |


### Frozen run summary (job-poster NER)

- Corpus size: **6327**
- Split: **5061 / 632 / 634** (train / val / test, seed 42)
- Test micro F1: **0.85** (seqeval ~**0.854**)
