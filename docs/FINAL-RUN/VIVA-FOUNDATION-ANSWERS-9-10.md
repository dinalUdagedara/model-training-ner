## Keyword Meanings (Simple)

- **Critical evaluation:** judging what worked, what did not, and why (with evidence).
- **Traceability:** linking requirements to implementation and testing evidence.
- **Achieved / Partial:** fully done vs partly done within scope.
- **Generalization:** whether results likely hold outside your specific dataset/scope.
- **Design-science style:** build artifact -> test -> reflect -> improve.
- **Directional finding:** useful insight, but not statistical proof for all populations.
