Proposed neural architecture — BiLSTM–CRF for résumé NER (Path 2)

Figure for design documentation. Hyperparameters follow the Path~2 training notebook (two-layer BiLSTM, Word2Vec embeddings, thirteen BIO labels). Alternative encoder configurations (e.g. BERT-based) may be documented in a separate figure.

Tensor dimensions in the proposed architecture
Component Shape Description
Token indices [B, L] Integer tensor; padding index 0 (convention used in training)
Embedding output [B, L, 256] Floating-point embeddings
BiLSTM output [B, L, 384] 192 features per direction, concatenated
Emission scores [B, L, K] Per-label scores prior to CRF normalisation
CRF transitions [K, K] Learned transition parameters
